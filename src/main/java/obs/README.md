# Online Book Store - Functional Programming

Complete refactoring of Online Book Store using Java 8+ functional paradigms.

## Features

- **Immutable Data Structures**: Thread-safe by design
- **Pure Functions**: No side effects, deterministic behavior
- **Stream API**: Declarative operations
- **Functional Composition**: Chainable operations
- **Error Handling**: Result<T> and Optional<T>
- **Currying**: Breaking functions into single-argument sequences
- **Partial Application**: Pre-filling arguments
- **Recursion**: Elegant recursive implementations

## Quick Start

```bash
mvn clean install
mvn test
mvn exec:java -Dexec.mainClass="obs.FunctionalExamples"
